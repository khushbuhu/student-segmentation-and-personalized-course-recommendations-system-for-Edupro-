# student-segmentation-and-personalized-course-recommendations-system-for-Edupro-
Student Segmentation and Personalized Course Recommendation System for EduPro using Power BI, Power Query, DAX, and Excel. The project analyzes learner behavior, segments learners into meaningful groups, and provides personalized course recommendations to improve engagement and learning outcomes.
## 📊 Key Insights
Key Insights
Casual Learners represent the largest learner segment.
Explorers demonstrate the highest learning engagement.
Data Science is the most popular course category.
Personalized recommendations improve course discovery and learner engagement.
## Tools & Technologies
Power BI
Power Query
DAX
Microsoft Excel
